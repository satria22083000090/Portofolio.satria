Tentang Saya

Halo, perkenalkan saya Muhammad Satria Pamungkas, seorang mahasiswa S1 Sistem Informasi Universitas Merdeka Malang. Saat ini saya sedang menekuni bidang pengembangan aplikasi berbasis web dengan tujuan menjadi seorang Fullstack Developer yang handal.

Ketertarikan saya pada dunia teknologi dimulai sejak lama, terutama dalam memahami bagaimana sebuah sistem informasi dapat membantu mempermudah kehidupan sehari-hari. Karena itu, saya terus berusaha memperdalam pengetahuan dalam pemrograman, basis data, desain antarmuka, hingga implementasi teknologi terbaru seperti React.js, Node.js, dan API.

Selain kuliah, saya juga aktif mengembangkan berbagai mini project sebagai sarana latihan, mulai dari pembuatan website portfolio, aplikasi akademik sederhana, hingga eksplorasi aplikasi mobile. Saya percaya bahwa konsistensi, kerja keras, dan rasa ingin tahu merupakan kunci utama untuk berkembang di dunia teknologi yang dinamis ini.

Harapan saya ke depan adalah dapat menjadi seorang developer yang tidak hanya mahir dalam teknologi backend dan frontend, tetapi juga mampu memberikan solusi nyata bagi masyarakat melalui inovasi digital.
